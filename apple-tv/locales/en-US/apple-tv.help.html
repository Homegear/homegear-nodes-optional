Automatically connects to an Apple TV in the current network, sends commands to it and listens for events.

Inputs

PWR
Turns the Apple TV off or on.
RMT
Sends remote control commands to your Apple TV. You can send the following commands as string in $message['payload']:
  • down: Down key
  • home: Home key
  • home_hold: Hold home key
  • left: Left key
  • menu: Menu key
  • next: Next key
  • pause: Pause key
  • play: Play key
  • play_pause: Toggle between play and pause
  • previous: Previous key
  • right: Right key
  • select: Select key
  • set_position: Seek to a playback position. The position must be provided as integer in $message['position'].
  • set_repeat: Set repeat setting. The setting must be provided as string in $message['state']. Possible values are all, off and track.
  • set_shuffle: Set shuffle setting. The setting must be provided as string in $message['state']. Possible values are albums, off and song.
  • skip_backward: Skips a time interval backward. The time interval is defined by the current app, but is typically between 15-30s.
  • skip_forward: Skips a time interval forward. The time interval is defined by the current app, but is typically between 15-30s.
  • stop: Stop key
  • suspend: Suspend the device.
  • top_menu: Go to main manu (same as long pressing the menu key)
  • up: Up key
  • volume_down: Volume down key
  • volume_up: Volume up key
  • wakeup: Wake up the device

Outputs

INIT
Outputs changes of the connection state.
PWR
Outputs true when the Apple TV is switched on and false when it is switched off.
PLAY
Outputs the current playback state as string. Possible values are idle, loading, paused, playing, seeking and stopped.
CONT
Outputs current playback information as a struct on changes.
